# OpenStreetMap Street Address Parser

This NodeJS library helps convert street address strings into a standard OSM `addr:*` format.

## Quickstart

```javascript
osmify('90 SPARKS STp
/*
{
  'addr:housenumber': '90',
  'addr:street': 'Sparks Street'
}
*/
```
